# 팬도리 🐼

## 🔧 개발환경 세팅

```bash
$ git clone https://github.com/Ahyoung-Kim/Fandolee.git
$ cd Fandolee
$ npm install
```

## 💚 실행 명령어

```bash
$ npm run start
또는
$ npm start
```
